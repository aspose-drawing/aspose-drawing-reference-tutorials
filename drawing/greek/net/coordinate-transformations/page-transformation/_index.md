# Μετασχηματισμός Συστήματος Συντεταγμένων (Μετασχηματισμός Σελίδας) στο Aspose.Drawing για .NET

## Εισαγωγή

Καλώς ήρθατε! Σε αυτό το tutorial θα ανακαλύψετε **πώς να εκτελέσετε έναν μετασχηματισμό συστήματος συντεταγμένων** σε μια σελίδα χρησιμοποιώντας το Aspose.Drawing για .NET. Είτε χρειάζεστε να **σχεδιάσετε αντικείμενα bitmap ορθογωνίου**, να κλιμακώσετε σχέδια, είτε απλώς να αντιστοιχίσετε μονάδες σελίδας σε μονάδες συσκευής, τα παρακάτω βήματα θα σας καθοδηγήσουν σε όλη τη διαδικασία—σαφή, περιεκτική και έτοιμη για αντιγραφή‑επικόλληση στο πρόγραμμά σας.

## Γρήγορες Απαντήσεις
- **Ποια είναι η κύρια κλάση για σχεδίαση;** `Graphics` από το Aspose.Drawing.
- **Πώς ορίζω τις μονάδες σελίδας;** Χρησιμοποιήστε `graphics.PageUnit = GraphicsUnit.Inch;`.
- **Μπορώ να σχεδιάσω ένα ορθογώνιο σε μετασχηματισμένη σελίδα;** Ναι—δημιουργήστε ένα `Pen` και καλέστε `DrawRectangle`.
- **Πού αποθηκεύεται το αποτέλεσμα;** Στο φάκελο που καθορίζετε στο `bitmap.Save`.
- **Χρειάζομαι άδεια για παραγωγική χρήση;** Απαιτείται έγκυρη άδεια Aspose.Drawing για εμπορική χρήση.

## Τι είναι ο μετασχηματισμός συστήματος συντεταγμένων;

Ένας **μετασχηματισμός συστήματος συντεταγμένων** αλλάζει τον τρόπο με τον οποίο οι λογικές συντεταγμένες σελίδας (όπως ίντσες ή χιλιοστά) αντιστοιχίζονται στις φυσικές συντεταγμένες συσκευής (pixel). Με την προσαρμογή του μετασχηματισμού, ελέγχετε την κλιμάκωση, την περιστροφή και τη μετάθεση όλων των λειτουργιών σχεδίασης, καθιστώντας ευκολότερο το σχεδιασμό γραφικών ανεξάρτητων από την ανάλυση.

## Γιατί να χρησιμοποιήσετε το Aspose.Drawing για μετασχηματισμούς σελίδας;

- **Πλήρης συμβατότητα με .NET** – λειτουργεί με .NET Framework, .NET Core και .NET 5/6.
- **Πλούσιο API γραφικών** – προσφέρει την ίδια λειτουργικότητα με το System.Drawing.Common χωρίς περιορισμούς πλατφόρμας.
- **Απλή διαχείριση μονάδων** – εναλλαγή μεταξύ ιντσών, χιλιοστών, σημείων κ.λπ., με μία μόνο ιδιότητα.
- **Υψηλής ποιότητας έξοδος** – δημιουργήστε PNG, JPEG, BMP και άλλες μορφές με ακριβή έλεγχο.

## Προαπαιτούμενα

Πριν προχωρήσουμε, βεβαιωθείτε ότι έχετε:

- **Βιβλιοθήκη Aspose.Drawing** – κατεβάστε την τελευταία έκδοση από την επίσημη ιστοσελίδα [εδώ](https://releases.aspose.com/drawing/net/).
- **Περιβάλλον Ανάπτυξης** – Visual Studio (οποιαδήποτε έκδοση) ή άλλο IDE .NET.
- **Δικαίωμα Εγγραφής** – έναν φάκελο στο μηχάνημά σας όπου θα αποθηκευτεί η μετασχηματισμένη εικόνα (αντικαταστήστε το `"Your Document Directory"` στον κώδικα με την πραγματική διαδρομή).

Τώρα που είμαστε έτοιμοι, ας περάσουμε από κάθε βήμα.

## Εισαγωγή Χώρων Ονομάτων

Πρώτα, φέρτε το απαιτούμενο namespace στο πεδίο ορατότητας:

```csharp
using System.Drawing;
```

> *Γιατί είναι σημαντικό*: Το `System.Drawing` περιέχει τα `Bitmap`, `Graphics`, `Pen` και τις δομές χρώματος που θα χρησιμοποιήσουμε σε όλο το tutorial.

## Βήμα 1: Δημιουργία Bitmap

Δημιουργήστε έναν κενό καμβά που θα φιλοξενήσει το μετασχηματισμένο σχέδιο. Καθορίζουμε μέγεθος **1000 × 800 pixel** και μορφή pixel που υποστηρίζει διαφάνεια alpha.

```csharp
Bitmap bitmap = new Bitmap(1000, 800, System.Drawing.Imaging.PixelFormat.Format32bppPArgb);
```

> Αυτό το bitmap λειτουργεί ως επιφάνεια σχεδίασης. Η επιλεγμένη μορφή pixel (`Format32bppPArgb`) εξασφαλίζει υψηλής ποιότητας απόδοση με προπολλαπλασιασμένο alpha.

## Βήμα 2: Δημιουργία Αντικειμένου Graphics

Ένα αντικείμενο `Graphics` παρέχει τις μεθόδους σχεδίασης (γραμμές, σχήματα, κείμενο). Το λαμβάνουμε από το bitmap που μόλις δημιουργήσαμε.

```csharp
Graphics graphics = Graphics.FromImage(bitmap);
```

> Το αντικείμενο `Graphics` είναι ο πυρήνας όλων των λειτουργιών απόδοσης· σέβεται τις ρυθμίσεις μετασχηματισμού που θα εφαρμόσουμε στη συνέχεια.

## Βήμα 3: Καθαρισμός Καμβά

Πριν σχεδιάσετε οτιδήποτε, καθαρίστε τον καμβά με ένα ουδέτερο χρώμα φόντου (γκρι σε αυτό το παράδειγμα). Αυτό το βήμα δείχνει επίσης πώς να γεμίσετε ολόκληρο το bitmap με ένα στερεό χρώμα.

```csharp
graphics.Clear(Color.FromKnownColor(KnownColor.Gray));
```

## Βήμα 4: Ορισμός Μετασχηματισμού (Εφαρμογή Μετασχηματισμού Σελίδας)

Εδώ **εφαρμόζουμε τον μετασχηματισμό σελίδας** ορίζοντας το `PageUnit` σε ίντσες. Αυτό λέει στη μηχανή γραφικών να ερμηνεύει όλες τις επόμενες συντεταγμένες ως ίντσες αντί για pixel.

```csharp
graphics.PageUnit = GraphicsUnit.Inch;
```

> *Συμβουλή*: Η αλλαγή του `PageUnit` είναι ένας απλός τρόπος να **μετασχηματίσετε τις συντεταγμένες της σελίδας** χωρίς να υπολογίζετε χειροκίνητα τιμές pixel.

## Βήμα 5: Σχεδίαση Ορθογωνίου (Draw rectangle bitmap)

Τώρα σχεδιάζουμε ένα ορθογώνιο που μετρά **1 ίντσα × 1 ίντσα** στη θέση (1, 1) ίντσες. Το πλάτος του `Pen` ορίζεται σε `0.1f` ίντσες, δίνοντας μια λεπτή αλλά ορατή γραμμή.

```csharp
Pen pen = new Pen(Color.FromKnownColor(KnownColor.Blue), 0.1f);
graphics.DrawRectangle(pen, 1, 1, 1, 1);
```

> Αυτό δείχνει το **draw rectangle bitmap** μετά την εφαρμογή του μετασχηματισμού—σημειώστε ότι το μέγεθος του ορθογωνίου ορίζεται σε ίντσες, όχι σε pixel.

## Βήμα 6: Αποθήκευση Εικόνας

Τέλος, αποθηκεύστε το μετασχηματισμένο bitmap στο δίσκο. Αντικαταστήστε το `"Your Document Directory"` με την πραγματική διαδρομή φακέλου στο μηχάνημά σας.

```csharp
bitmap.Save("Your Document Directory" + @"CoordinateSystemsTransformations\PageTransformation_out.png");
```

> Το αρχείο εξόδου (`PageTransformation_out.png`) θα περιέχει το ορθογώνιο που σχεδιάστηκε χρησιμοποιώντας τον μετασχηματισμό συστήματος συντεταγμένων που ορίσαμε νωρίτερα.
